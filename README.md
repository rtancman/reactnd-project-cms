# CMS Project

## Quick Overview
```bash
git clone git@github.com:rtancman/reactnd-project-cms.git
make setup
make run
```

## Installation
### Clone repository

```sh
git clone git@github.com:rtancman/reactnd-project-cms.git
```

### Install dependencies
```sh
npm install

or

yarn install
```

### Run project 
```sh
npm start

or

yarn start
```
